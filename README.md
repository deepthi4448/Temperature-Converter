# 🌡️ Temperature Converter

A simple and responsive **Temperature Converter** built using **HTML, CSS, and JavaScript**. This application allows users to instantly convert temperatures between **Celsius (°C)**, **Fahrenheit (°F)**, and **Kelvin (K)** with real-time calculations.

## 🚀 Features

- 🌡️ Convert temperatures between Celsius, Fahrenheit, and Kelvin
- ⚡ Instant conversion on user input
- 📱 Responsive and user-friendly interface
- 🎨 Clean and modern UI
- 💻 Built using Vanilla JavaScript (No external libraries)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📖 How It Works

1. Enter a temperature value in any one of the input fields.
2. The application automatically converts the value into the other two temperature units.
3. The results are displayed instantly without refreshing the page.

## 📐 Temperature Conversion Formulas

### Celsius → Fahrenheit
```
(°C × 9/5) + 32
```

### Celsius → Kelvin
```
°C + 273.15
```

### Fahrenheit → Celsius
```
(°F − 32) × 5/9
```

### Fahrenheit → Kelvin
```
((°F − 32) × 5/9) + 273.15
```

### Kelvin → Celsius
```
K − 273.15
```

### Kelvin → Fahrenheit
```
((K − 273.15) × 9/5) + 32
```
